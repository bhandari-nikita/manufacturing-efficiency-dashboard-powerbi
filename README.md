# Manufacturing Efficiency Dashboard (Power BI)

📊 Interactive Power BI dashboard for monitoring manufacturing performance.

## Dashboard Preview

(image)

## Business Problem

Manufacturing teams need a centralized dashboard to monitor
production, downtime, defect rates, and machine performance.

## KPIs

• Total Units Produced
• Total Defects
• Defect Rate
• Total Downtime

## Dashboard Features

✔ Interactive slicers
✔ DAX Measures
✔ Trend Analysis
✔ Machine Performance
✔ Product-wise Analysis
✔ Shift-wise Defect Analysis

## Tools Used

• Power BI Desktop
• Power Query
• DAX
• CSV Dataset

## Dataset

Synthetic manufacturing dataset created for portfolio purposes.
